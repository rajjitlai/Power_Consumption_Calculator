# Power Consumption Calculator

A sleek, responsive, and modern web application to calculate the power consumption and energy costs of your electrical appliances. Built with a beautiful dark-themed glassmorphism design.

![Power Consumption Calculator Preiew](/assets/preview.png)

## Features

*   **Modern Glassmorphism UI:** A visually stunning dark theme with blurred, glass-like UI elements and animated background shapes.
*   **Comprehensive Calculations:** Calculates both daily and monthly energy consumption in Kilowatt-hours (kWh).
*   **Cost Estimation:** Input your local electricity cost per unit to get an estimated monthly cost.
*   **Fully Responsive:** Works perfectly on desktops, tablets, and mobile devices.
*   **Print Support:** Includes dedicated CSS for printing clean, ink-saving reports of your calculations.
*   **Data Guide:** Built-in information section to help users find appliance wattage and electricity rates.
*   **No Dependencies:** Built entirely with Vanilla HTML, CSS, and JavaScript.

## Usage

1.  Open `index.html` in your web browser.
2.  Enter the name of your appliance (e.g., "Air Conditioner").
3.  Enter the quantity and the power rating in Watts.
4.  Specify how many hours a day and days a month the appliance is used.
5.  (Optional) Enter your local cost per kWh to see cost estimates.
6.  Click **Calculate** to see the results.
7.  Click **Print Report** to save or print a clean copy of the results.
